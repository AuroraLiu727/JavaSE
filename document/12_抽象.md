# 抽象的介绍
1. **抽象类怎么来的：**抽取共性方法,放到父类中,发现方法没法实现,因为每个子类对此方法的实现方式细节不一样，此时方法体说不清道不明,可以定义成抽象方法
2. **关键字**：`abstract`
3. **抽象类定义方式**：`public abstract class 类名{}`
4. **抽象方法定义方式**：`修饰符 abstract 返回值类型 方法名(参数);`
5. **怎样实现抽象方法**：子类继承抽象类之后重写抽象类中的抽象方法，具体实现细节看子类怎样重写方法体
6. **示例**

**抽象类（Animal）**

```java
package com.aurora.abstractDemo.basicUse;

public abstract class Animal {
    /*
        抽象类
     */
    public abstract void eat();
    public abstract void drink();
}

```

**子类（Dog）**

```java
package com.aurora.abstractDemo.basicUse;

public class Dog extends Animal {
    @Override
    public void eat() {
        System.out.println("狗啃骨头");
    }

    @Override
    public void drink() {
        System.out.println("狗喝水");
    }
}

```

**子类（Cat）**

```java
package com.aurora.abstractDemo.basicUse;

public class Cat extends Animal {
    @Override
    public void eat() {
        System.out.println("猫吃鱼");
    }

    @Override
    public void drink() {
        System.out.println("猫喝水");
    }
}

```

**测试类**

```java
package com.aurora.abstractDemo.basicUse;

public class Test01 {
    public static void main(String[] args) {
        Dog dog = new Dog();
        dog.eat();
        dog.drink();

        System.out.println("=======================");

        Cat cat = new Cat();
        cat.eat();
        cat.drink();
    }
}

```

7. **注意**
+ 抽象方法所在的类一定是抽象类（上例中抽象方法`eat()`和`drink()`在抽象类`Animal`中）
+ 抽象类中不一定非得有抽象方法（上例的`Animal`类中可以不写方法）
+ 子类继承父类之后，需要重写父类中的所有抽象方法，不然会编译报错，除非该子类也是抽象类（上例中`Dog`类和`Cat`类继承`Animal`之后必须重写`Animal`中的所有抽象方法，即`eat()`方法和`drink()`方法）
+ 抽象类不能`new`对象，只能通过`new`子类对象调用重写方法（上例中在子类`Dog`和`Cat`中重写方法，在测试类中`new`子类对象`Dog`和`Cat`，并分别调用重写后的方法）

# 抽象的注意事项
1. 抽象类中可以有**成员变量**，**构造方法**（供子类创建对象时，初始化父类属性使用的（在子类的构造方法中使用`super()`）），**成员方法**
2. 示例

**父类**

```java
package com.aurora.abstractDemo.Demo;

public abstract class Employee {
    private String name;
    private int age;

    public Employee() {
    }

    public Employee(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }

    //抽象方法
    public abstract void work();
}

```

**子类**

```java
package com.aurora.abstractDemo.Demo;

public class Teacher extends Employee{
    public Teacher() {
    }

    public Teacher(String name, int age) {
        super(name, age);
    }

    @Override
    public void work() {
        System.out.println(this.getName() + " is working...");
    }
}

```

**测试类**

```java
package com.aurora.abstractDemo.Demo;

public class Test {
    public static void main(String[] args) {
        Teacher teacher = new Teacher("张三", 20);
        System.out.println(teacher.getName() + "-" + teacher.getAge());
        teacher.work();
    }
}

```
