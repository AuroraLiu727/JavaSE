# 常用的DOS命令
1. 打开dos命令窗口：

```markdown
win + R -> 输入cmd -> 回车
```

2. 常用的dos命令

| 作用 | 命令 |
| --- | --- |
| 切换盘符 | 盘符名: -> 回车   盘符名不区分大小写,但是计算机上必须有指定的盘符 |
| 查看当前路径下的文件或者文件夹 | dir |
| 进入到指定文件夹下 | cd 文件夹名字 |
| 进入到多级文件夹下 | cd 文件夹名字\文件夹名字 |
| 退到上一级目录 | cd..或者 cd .. |
| 直接退到磁盘位置(退到根目录) | cd\或者cd \ |
| 清屏 | cls |
| 退出黑窗口 | exit |
| 创建文件夹 | mkdir 文件夹名 |
| 创建多级文件夹 | mkdir 文件夹名\文件夹名 |
| 删除文件夹 | rd 文件夹名   注意:删除的文件夹必须是空的   不走回收站 |
| 删除文件 | del 文件名.后缀名   不走回收站 |
| 批量删除文件 | del *.后缀名 |
