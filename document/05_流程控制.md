# 键盘录入Scanner
1. **介绍：**java定义好的一个类，功能是将数据通过键盘录入的方式放到代码中运行，位置在`java.util`
2. 使用方式：
+ 导包：`import java.util.Scanner`
+ 创建对象：`Scanner 变量名 = new Scanner(System.in);`
+ 调用方法，实现键盘录入
    - 输入整数`Int`型的：`变量名.nextInt()`，**遇到空格、Tab或回车等空白符结束录入**
    - 输入字符串 `String`型的`变量名.next()`，**遇到空格、Tab或回车等空白符结束录入**
    - 要想录入一整行包括空白符的数据，可以使用`变量名.nextLine()`
3. 使用示例

```java
package com.aurora.ProcessControl;

import java.util.Scanner;

public class ScannerDemo01 {
    /*
        键盘录入使用示例
     */
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        //录入int型整数
        int data1 = sc.nextInt();
        System.out.println("data1 = " + data1);

        //录入String型字符串
        String data2 = sc.next();
        System.out.println("data2 = " + data2);
    }
}

```

# Random随机数
1. 作用：在指定的范围内指定一个整数
2. 使用：
+ 导包：`import java.util.Random`
+ 创建对象：`Random 变量名 = new Random()`
+ 调用方法，生成随机数：
    - `变量名.nextInt()`->在`int`的取值范围内随机一个整数
    - 在指定范围内生成随机数：`nextInt(int bound)`->在`0 ~ bound - 1`之间取一个随机数
    - 不从0开始随机：`nextInt(10) + 1`->在`1~10`之间随机一个数
3. 使用示例

```java
package com.aurora.ProcessControl;

import java.util.Random;

public class RandomDemo01 {
    /*
        Random的使用示例
     */
    public static void main(String[] args) {
        Random random = new Random();
        
        //在int范围内生成随机数
        int data1 = random.nextInt();
        System.out.println("data1 = " + data1);

        //在0-10之间生成随机数
        int data2 = random.nextInt(10);
        System.out.println("data2 = " + data2);

        //在100-999之间生成随机数
        int data3 = random.nextInt(900) + 100;
        System.out.println("data3 = " + data3);
    }
}

```

# switch选择语句
## 基本使用
1. 格式

```java
switch(变量){
    case 常量值1:
        执行语句1；
        break;
    case 常量值1:
        执行语句1；
        break;
    case 常量值1:
        执行语句1；
        break;
    case 常量值1:
        执行语句1；
        break;
    ...
    default:
        执行语句n;
        break;
}
```

2. `switch`能匹配的数据类型：`byte`、`short`、`int`、`char`、`枚举类型`、`String类型`
3. 使用示例

```java
package com.aurora.ProcessControl;

import java.util.Scanner;

public class switchDemo01 {
    /*
        switch基本用法
     */
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.println("请您输入一个整数：");
        int data = sc.nextInt();
        switch (data) {
            case 1:
                System.out.println("a");
                break;
            case 2:
                System.out.println("b");
                break;
            case 3:
                System.out.println("c");
                break;
            default:
                System.out.println("d");
                break;
        }
    }
}

```

## case的穿透性
### case穿透性的定义
1. 定义：如果没有`break`，就会出现`case`的穿透性，程序就一直往下穿透执行，知道遇到了`break`或者`switch`代码执行结束
2. 示例

```java
package com.aurora.ProcessControl;

import java.util.Scanner;

public class switchDemo02 {
    /*
        case的穿透性
     */
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int data =  sc.nextInt();
        switch (data) {
            case 1:
                System.out.println("a");
            case 2:
                System.out.println("b");
            case 3:
                System.out.println("c");
            case 4:
                System.out.println("d");
                break;
            default:
                System.out.println("end");
                break;
        }
    }
}

```

# 分支语句
1. if的三种形式

```java
//第一种
if(boolean表达式){
    执行语句;
}

//第二种
if(boolean表达式){
    执行语句1;
} else {
    执行语句2;
}

//第三种
if(boolean表达式){
    执行语句1;
}else if{
    执行语句2;
}else if{
    执行语句3;
}...else{
    执行语句n;
}
```

2. 示例

根据最新的年龄段划分标准:

  0-6岁为婴幼儿

  7-12岁为少儿

  13-17岁为青少年

  18-45岁为青年

  46-69岁为中年

  69岁以上为老年

请键盘录入一个年龄,判断属于什么年龄段

```java
package com.aurora.ProcessControl;

import java.util.Scanner;

public class ifelseDemo01 {
    /*
        利用if else语句进行年龄划分
     */
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int age = sc.nextInt();

        if (age <0 || age >130){
            System.out.println("年龄不符合实际");
        }else{
            if (age >= 0 && age <= 6){
                System.out.println("婴幼儿");
            }else if(age >=7 && age <= 12){
                System.out.println("少儿");
            }else if(age >= 13 && age <=17){
                System.out.println("青少年");
            }else if(age >= 18 && age <= 45){
                System.out.println("青年");
            }else if (age >=46 && age <= 69){
                System.out.println("中年");
            }else {
                System.out.println("老年");
            }
        }
    }
}
```

## while循环
1. 格式

```java

    步进表达式;
}
```

2. 示例:统计1-100之间偶数的个数

```java
package com.aurora.ProcessControl;

public class whileDemo01 {
    /*
        while循环的使用示例：统计1-100之间偶数的个数
     */
    public static void main(String[] args) {
        int i = 1, count = 0;
        while ( i <= 100) {
            if (i % 2 == 0) {
                count++;
            }
            i++;
        }
        System.out.println("count = " + count);
    }
}

```

# 练习
1. 创建Scanner和Random对象
2. 调用Random中的nextInt(100)+1在1-100之间随机一个数  rdNumber
3. 调用Scanner中的nextInt()方法 键盘录入一个要猜的数   scNumber
4. 如果scNumber大于rdNumber,证明猜大了
5. 如果scNumber小于rdNumber,证明猜小了
6. 如果scNumber等于rdNumber,证明猜中了

```java
package com.aurora.ProcessControl;

import java.util.Random;
import java.util.Scanner;

public class Practice01 {
    /*
        流程控制练习：猜数字小游戏
     */
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        Random rand = new Random();
        int data = rand.nextInt(100) + 1;
        System.out.println("请输入您要猜的数字：");
        int user_num = sc.nextInt();
        while(true){
            if(user_num == data){
                System.out.println("猜对了！");
                break;
            }else if(user_num < data){
                System.out.println("猜小了，请再次输入您要猜的数字：");
                user_num = sc.nextInt();
            }else{
                System.out.println("猜大了，请再次输入您要猜的数字：");
                user_num = sc.nextInt();
            }
        }
    }
}

```
