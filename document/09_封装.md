# 封装的介绍及使用
1. 什么是封装思想：隐藏对象内部的复杂性,只对外提供公开,公共的接口,便于外界调用,从而提高了系统的可扩展性,可维护性,安全性,通俗来说,把该隐藏的隐藏起来(细节),把该暴露的暴露出来(对外提供的供别人使用的接口),这就是封装思想
2. 定义的成员变量，只要是`new`出来的对象，就可以随便调用，随便赋值，哪怕是不合理的值我们也挡不住，怎么解决？
+ 关键字：`private`(私有化的)，被`private`修饰的成员只能在本类中使用，再别的类中使用不了。
+ `private`的使用：
    - 修饰成员变量：`private 数据类型 变量名;`
    - 修饰方法：将`public`改成`private`，其他的都一样
+ 注意：将代码放到一个方法中，是封装的体现，一个成员被`private`修饰也是封装的体现
3. 属性被私有化了，外界直接调用不了，那么此时属性就不能直接赋值取值了，所以需要提供公共的接口：`get/set`方法
+ `set`方法：为属性赋值
+ `get`方法：获取属性值
4. 示例

```java
package com.aurora.OOP.fengzhuang;

/*
    封装的使用示例
 */
public class Person {
    private String name;
    private int age;

    //为name提供get/set方法
    public void setName(String mingzi){
        name = mingzi;
    }
    public String getName(){
        return name;
    }

    //为age提供get/set方法
    public void setAge(int nianling){
        if(nianling < 0 || nianling > 150){
            System.out.println("岁数不符合实际");
        }else{
            age = nianling;
        }
    }
    public int getAge(){
        return age;
    }
}

```

```java
package com.aurora.OOP.fengzhuang;

public class PersonDemo01 {
    /*
        封装的使用示例
     */
    public static void main(String[] args) {
        Person person = new Person();
        person.setName("liu");
        person.setAge(21);

        String name = person.getName();
        int age = person.getAge();
        System.out.println(name);
        System.out.println(age);
    }
}

```

# this的介绍
+ `this`概述：`this`代表的当前对象（哪个对象调用的`this`所在的方法，`this`就代表哪个对象）
+ 作用：在成员变量和局部变量重名时使用(例如当前对象的属性和某个方法传入的参数都叫`name`)，`this`可以区分重名的成员变量和局部变量，`this`点出来的一定是成员的变量
1. 示例1（哪个对象调用的`this`所在的方法，`this`就代表哪个对象）

```java
package com.aurora.OOP.fengzhuang.thisDemo;

public class Person {
    String name;

    public void speak(String name){
        System.out.println(this);
        System.out.println(name + "您好，我是" + this.name);
    }
}

```

```java
package com.aurora.OOP.fengzhuang.thisDemo;

public class PersonDemo01 {
    /*
        哪个对象调用的this所在的方法，this就代表哪个对象
     */
    public static void main(String[] args) {
        Person person1 = new Person();
        System.out.println(person1);
        person1.name = "张三";
        person1.speak("李四");


        Person person2 = new Person();
        System.out.println(person2);
        person2.name = "王五";
        person2.speak("赵六");

    }
}

```

+ `Person`类中`speak`方法的`System.out.println(this);`语句输出的`this`的地址，与`PersonDemo01`类中`System.out.println(person1);`语句输出的`person`的地址相同，所以就印证了**哪个对象调用的**`**this**`**所在的方法，**`**this**`**就代表哪个对象。**
+ 可见`person1`与`person2`是两个不同的对象，他们指向不同的地址
+ 在对象`person1`中，`this.name = "张三"`，`speak`方法输入的参数`name = "李四"`
2. 示例：使用`this`关键字构造`get/set`方法

# 构造方法
1. 定义：方法名与类名一致且能初始化对象的方法
2. 分类：
+ 无参构造：没有参数
+ 有参构造：有参数，参数是为指定的属性赋值
+ 满参构造：给所有属性赋值
3. 特点：
+ 方法名和类名一致
+ 没有返回值，**连**`**void**`**都没有**

## 空参构造
1. 格式

```java
public 类名(){
    
}
```

2. 作用：`new`对象使用
3. 特点：每个类中默认都有一个无参构造，不写也有，`jvm`会自动提供
4. 使用：`new`对象就是在调用空参构造方法

## 有参构造
1. 格式

```java
public 类名(形参){
    为属性赋值
}
```

2. 作用
+ `new`对象
+ 为属性赋值
3. `jvm`不会自动提供有参构造，但是将有参构造写出来，`jvm`不再提供无参构造，所以需要有参、无参的构造都写上去

## 构造方法示例
```java
package com.aurora.OOP.fengzhuang.Constructor;

public class Person {
    private String name;
    private int age;

    //无参构造
    public Person(){
        System.out.println("无参构造");
    }

    //有参构造
    public Person(String name, int age){
        this.name = name;
        this.age = age;
    }

    public String getName(){
        return  this.name;
    }
    public int getAge(){
        return this.age;
    }
    public void setName(String name){
        this.name = name;
    }
    public void setAge(int age){
        this.age = age;
    }
}

```

```java
package com.aurora.OOP.fengzhuang.Constructor;

public class PersonDemo01 {
    /*
        构造方法：有参构造与无参构造
     */
    public static void main(String[] args) {
        //无参构造的使用
        Person person1 = new Person();
        person1.setName("张三");
        person1.setAge(18);
        System.out.println(person1.getName());
        System.out.println(person1.getAge());

        //有参构造的使用
        Person person2 = new Person("李四", 20);
        System.out.println(person2.getName());
        System.out.println(person2.getAge());
    }
}

```

# 标准JavaBean
## JavaBean介绍
1. 什么是JavaBean：JavaBean是Java语言编写类的一种标准规范
2. 符合JavaBean的类的要求：
+ 类必须是具体的（非抽象abstract）和公共的，`public class 类名`
+ 具有无参构造方法和有参构造方法
+ 成员变量私有化，并提供用来操作成员变量的`get/set`方法
3. 标准格式示例

```java
package com.aurora.OOP.fengzhuang.JavaBean;

public class Person {
    private String name;
    private int age;

    public Person() {
    }

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }
}

```

## JavaBean怎么来的
将来的javabean都是和数据库的表相关联

+ 类名 -> 表名
+ 属性名 -> 列名
+ 对象 -> 表中每一行数据
+ 属性值 -> 表中单元格中的数据
