# 数组的定义
1. 数组概述：是一个数据容器，数组本身属于引用数据类型，可以一次存储多个数据
2. 特点
+ 既能存储基本类型的数据，也能存储引用类型的数据
+ 定长：定义数组时长度为多长，就能存储多少个数据
3. 定义方式
+ 动态初始化：
    - `数据类型[] 数组名 = new 数据类型[长度]`
    - `数据类型 数组名[] = new 数据类型[长度]`
    - 等号右边的数据类型要和等号左边的数据类型保持一致
+ 静态初始化（不推荐使用）
    - `数据类型[] 数组名 = new 数据类型[]{元素1, 元素2, ...}`
    - `数据类型 数组名[] = new 数据类型[]{元素1, 元素2, ...}`
+ 简化的静态初始化（推荐使用）
+ `数据类型[] 数组名 = {元素1, 元素2, ...}`
4. 动态初始化和静态初始化的区别
+ 动态初始化：定义的时候只知道长度，没有具体的数据
+ 静态初始化：定义的时候已经知道要存的数据
5. 使用示例

```java
package com.aurora.Array;

public class ArrayDemo01 {
    /*
        数组的定义
     */
    public static void main(String[] args) {
        //动态初始化
        int[] arr1 = new int[3];
        String[] arr2 = new String[3];
        
        //静态初始化
        int[] arr3 = new int[]{1, 2, 3, 4, 5};
        
        //简化静态初始化
        int[] arr4 = {1, 2, 3, 4, 5};
        String[] arr5 = {"hello", "world"};
    }
}

```

# 数组操作
## 获取数组的长度
1. 格式：`数组名.length`
2. 注意：`length`后面不要带小括号，因为`length`不是一个方法，而是数组中的一个属性
3. 使用示例

```java
package com.aurora.Array;

public class ArrayDemo02 {
    /*
        获取数组长度的操作
     */
    public static void main(String[] args) {
        int[] arr = {1, 2, 3, 4, 5};
        int len = arr.length;
        System.out.println("len = " + len);
    }
}

```

## 索引
1. 概述：元素在数组中存储的位置，范围是`0 ~ arr.length-1`

## 存储元素
1. 格式：`数组名[索引值] = 值`
2. 使用示例

```java
package com.aurora.Array;

public class ArrayDemo03 {
    /*
        数组的存储元素操作
     */
    public static void main(String[] args) {
        int[] arr1 = new int[3];
        arr1[0] = 1;
        arr1[1] = 2;
        arr1[2] = 3;
    }
}

```

## 获取元素
1. 利用索引获取对应的数组元素：`数组名[索引值]`
2. 注意：
+ 如果直接输出数组名，会直接输出数组在内存中的地址值
+ 如果数组中没有存元素，那么直接获取索引上对应的元素也是有值的，只不过是数组中的元素默认值
    - 整数：`0`
    - 小数：`0.0`
    - 字符：`'\u0000'`->空白字符->对应的`int`值是0
    - 布尔：`false`
    - 引用：`null`
3. 使用示例

```java
package com.aurora.Array;

public class ArrayDemo04 {
    /*
        获取数组元素
     */
    public static void main(String[] args) {
        int[] arr1 = {1, 2, 3};
        System.out.println("arr1 = " + arr1);   //数组起始地址
        System.out.println("arr1[0] = " + arr1[0]); //数组对应索引处的值

        //空整形数组
        int[] arr2 = new int[3];
        System.out.println("arr2[0] = " + arr2[0]);

        //空浮点型数组
        double[] arr3 = new double[3];
        System.out.println("arr3[0] = " + arr3[0]);

        //空字符型数组
        char[] arr4 = new char[3];
        System.out.println("arr4[0] = " + arr4[0]);

        //空布尔型数组
        boolean[] arr5 = new boolean[3];
        System.out.println("arr5[0] = " + arr5[0]);

        //空字符串型（引用型）数组
        String[] arr6 = new String[3];
        System.out.println("arr6[0] = " + arr6[0]);
    }
}

```

## 数组扩容
1. 步骤：
+ 定义一个更长的新数组
+ 将旧数组的内容复制到新数组中
+ 将旧数组的地址给老数组
2. 示例

```java
package com.aurora.Array;

public class ArrayPractice08 {
    /*
        数组扩容
     */
    public static void main(String[] args) {
        int[] oldArr = {1, 2, 3, 4, 5};
        //创建更长的新数组
        int[] newArr = new int[10];

        //将旧数组的内容复制到新数组中
        for (int i = 0; i < oldArr.length; i++) {
            newArr[i] = oldArr[i];
        }

        //将新数组的地址给旧数组
        oldArr = newArr;

        System.out.println("旧数组的长度变为：" + oldArr.length);
        for (int i = 0; i < oldArr.length; i++) {
            System.out.print(oldArr[i] + " ");
        }
    }
}

```

## 合并数组
```java
package com.aurora.Array;

public class ArrayPractice09 {
    /*
        合并数组
     */
    public static void main(String[] args) {
        int[] arr1 = {1, 2, 3};
        int[] arr2 = {4, 5, 6};
        int[] newArr = new int[arr1.length + arr2.length];
        for (int i = 0; i < arr1.length; i++) {
            newArr[i] = arr1[i];
        }
        for (int i = 0; i < arr2.length; i++) {
            newArr[arr1.length + i] = arr2[i];
        }

        for (int i = 0; i < newArr.length; i++) {
            System.out.printf(newArr[i] + " ");
        }
    }
}

```

# 数组操作时的常见问题
## 数组越界异常_ArrayIndexOutOfBoundsException
例如，定义了一个数组`int[] arr = {1, 2, 3}`，明显数组的长度为3，只能访问`arr[0]`、`arr[1]`、`arr[2]`，访问其他的索引值均会导致数组越界异常，例如访问`arr[3]`、`arr[-1]`等。

```java
package com.aurora.Array;

public class ArrayDemo07 {
    /*
        数组越界异常
     */
    public static void main(String[] args) {
        int[] arr = {1, 2, 3};
        System.out.println(arr[0]);     //1
        System.out.println(arr[3]);     //ArrayIndexOutOfBoundsException
    }
}

```

## 空指针异常_NullPointerException
1. 原因：当一个对象为`null`时，再调用此对象中的其他成员

```java
package com.aurora.Array;

public class ArrayDemo08 {
    /*
        空指针异常示例
     */
    public static void main(String[] args) {
        int[] arr = new int[3];
        System.out.println(arr.length); //3

        arr = null;
        System.out.println(arr.length); //NullPointerException
    }
}

```

# 二维数组
## 二维数组的定义格式
1. 动态初始化
+ `数据类型[][] 数组名 = new 数据类型[m][n]`，**常用**
+ `数据类型 数组名[][] = new 数据类型[m][n]`
+ `数据类型[] 数组名[] = new 数据类型[m][n]`
+ `m`代表的是二维数组的长度，`n`代表的是二维数组中每个一维数组的长度
+ 若是`数据类型[][] 数组名 = new 数据类型[m][]`，则表示二维数组中的一维数组没有被创建
2. 静态初始化
+ `数据类型[][] 数组名 = new 数据类型[][]{{元素1, 元素2, ...}, {元素1, 元素2, ...}, {元素1, 元素2, ...}, ...}`
+ `数据类型 数组名[][] = new 数据类型[][]{{元素1, 元素2, ...}, {元素1, 元素2, ...}, {元素1, 元素2, ...}, ...}`
+ `数据类型[] 数组名[] = new 数据类型[][]{{元素1, 元素2, ...}, {元素1, 元素2, ...}, {元素1, 元素2, ...}, ...}`
3. 简化静态初始化
+ `数据类型[][] 数组名 = {{元素1, 元素2, ...}, {元素1, 元素2, ...}, {元素1, 元素2, ...}, ...}`
+ `数据类型 数组名[][] = {{元素1, 元素2, ...}, {元素1, 元素2, ...}, {元素1, 元素2, ...}, ...}`
+ `数据类型[] 数组名[] = {{元素1, 元素2, ...}, {元素1, 元素2, ...}, {元素1, 元素2, ...}, ...}`
4. 使用示例

```java
package com.aurora.Array;

public class ArrayDemo09 {
    /*
        二维数组的定义方式
     */
    public static void main(String[] args) {
        int[][] arr1 = new int[2][2];
        int[][] arr2 = new int[2][];
        
        int[][] arr3 = {{1, 2, 3}, {4, 5, 6}, {7, 8, 9}};
    }
}

```

## 获取二维数组的长度
1. 格式
+ 直接获取二维数组的长度：`数组名.length`
+ 获取二维数组中每个一维数组的长度，需要先遍历二维数组，将每个一维数组从二维数组中拿出来,`arr[i]`表示每个一维数组的长度
2. 示例

```java
package com.aurora.Array;

public class ArrayDemo10 {
    /*
        二维数组的长度
     */
    public static void main(String[] args) {
        int[][] arr = {{1, 2, 3}, {4, 5, 6}};

        //二维数组的长度
        System.out.println(arr.length);

        //二维数组中每个一维数组的长度
        for (int i = 0; i < arr.length; i++) {
            System.out.println(arr[i].length);  //arr[i]表示每个一维数组
        }
    }
}

```

## 二维数组的遍历
1. 格式

```java
for (int i = 0; i < arr.length; i++) {	//遍历二维数组中的每个一维数组
    for (int j = 0; j < arr[i].length; j++) {	//遍历一维数组中的每个元素
        System.out.print(arr[i][j] + " ");
}
```

2. 示例

```java
package com.aurora.Array;

public class ArrayDemo13 {
    /*
        二维数组遍历
     */
    public static void main(String[] args) {
        int[][] arr = {{1, 2, 3}, {4, 5, 6}, {7, 8, 9}};

        //遍历二维数组
        for (int i = 0; i < arr.length; i++) {
            for (int j = 0; j < arr[i].length; j++) {
                System.out.print(arr[i][j] + " ");
            }
        }
    }
}

```
