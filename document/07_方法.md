# 方法的使用
## 方法介绍
1. 概述：方法是拥有功能性代码的代码块
2. 作用：将不同功能放在不同的方法中，给每个方法取个名字，直接调用方法名，对应的方法就执行起来了，好维护
3. 通用定义格式

```java
修饰符 返回值类型 方法名(参数){
    方法体;
    return 结果;
}
```

4. 分类：分成四种方法来学习
+ 无参无返回值方法
+ 有参无返回值方法
+ 无参有返回值方法
+ 有参有返回值方法

## 无参无返回值方法
1. 格式

```java
public static void(){
    方法体;
}
```

2. 调用方式：直接调用，在其他方法中，通过`方法名()`调用。
3. 注意事项
+ `void`关键字代表无返回值，写了`void`，就不能在方法中写`return`结果
+ 方法不调用不执行，`main`方法是`jvm`调用的
+ 方法之间不能互相嵌套，方法之间是平级关系
+ 方法的执行顺序只和调用顺序有关
4. 关键字解释
+ public
    - 访问修饰符（Access Modifier）。
    - 表示这个方法是 公开的，在任何其他类中都可以直接调用。
    - 对比：
        * private：只能在当前类内部使用
        * protected：在同一个包内或子类中可用
    - （无修饰符，包访问权限）：仅同一个包内可用
+ static
    - 表示这个方法是 静态方法。
    - 静态方法属于 类本身，而不是某个对象。
    - 调用时不需要先创建对象，可以直接用 类名.方法名 调用。
    - 常见场景：main 方法就是静态的：

**后面还会详细学到，这里先了解**

5. 使用示例

```java
package com.aurora.method;

public class methodDemo01 {
    /*
        无参无返回值的方法定义与调用
     */
    public static void main(String[] args) {
        hello();
        world();
        hello();
    }

    public static void hello(){
        System.out.println("hello");
    }

    public static void world(){
        System.out.println("world");
    }
}

```

## 有参无返回值方法
1. 格式

```java
public static void 方法名(数据类型 变量名){
    方法体;
}
```

2. 调用方式：`方法名(具体的值);`
3. 使用示例：定义一个方法，实现两个整数相加

```java
package com.aurora.method;

public class methodDemo02 {
    /*
        有参数无返回值的方法示例：定义一个方法，实现两个整数相加
     */
    public static void main(String[] args) {
        add(10, 20);

    }

    public static void add(int a, int b){
        int sum = a + b;
        System.out.println("sum = " + sum);
    }
}

```

## 无参数有返回值方法
1. 格式

```java
public static 返回值类型 方法名(){
    方法体;
    return 结果;
}
```

2. 调用：
+ 打印调用(不推荐)：`sout(方法名())`
+ 赋值调用(推荐)：调用完之后用一个变量接收返回值结果，`数据类型 变量名 = 方法名()`
3. 使用示例：定义一个方法，实现两个整数相加，将结果返回

```java
package com.aurora.method;

public class methodDemo03 {
    /*
        无参数有返回值的方法使用示例：定义一个方法，实现两个整数相加，将结果返回
     */
    public static void main(String[] args) {
        int sum = add();
        System.out.println("sum = " + sum);
    }

    public static int add(){
        int a = 10;
        int b = 20;
        return a + b;
    }
}

```

## 有参数有返回值方法
1. 格式

```java
public static 返回值类型 方法名(参数){
    方法体;
    return 结果;
}
```

2. 调用
+ 打印调用(不推荐)：`sout(方法名(参数))`
+ 赋值调用(推荐)：调用完之后用一个变量接收返回值结果，`数据类型 变量名 = 方法名(参数)`
3. 使用示例：定义一个方法，实现两个整数相加，将结果返回

```java
package com.aurora.method;

public class methodDemo04 {
    /*
        有参数有返回值的方法使用示例：定义一个方法，实现两个整数相加，将结果返回
     */
    public static void main(String[] args) {
        int result = add(10, 20);
        System.out.println("result = " + result);
    }

    public static int add(int a, int b){
        return a + b;
    }
}

```

+ 形式参数（形参）：在定义方法的时候形式上定义的参数，此参数还没有值
+ 实际参数（实参）：在调用方法的时候给形参赋予的具体的值

## 变量作为实参使用
1. 示例：定义一个方法，比较两个整数的大小，如果第一个比第二个大，返回`true`，否则返回`false`

```java
package com.aurora.method;

public class methodDemo05 {
    /*
        变量作为实参使用：定义一个方法，比较两个整数的大小，如果第一个比第二个大，返回true，否则返回false
     */
    public static void main(String[] args) {
        int a = 10;
        int b = 20;
        boolean result = compare(a, b);
        System.out.println("result = " + result);
    }

    public static boolean compare(int a, int b){
        if(a > b){
            return true;
        }else{
            return false;
        }
    }
}

```

# 方法的注意事项
1. 方法不调用不执行
2. 方法的执行顺序只和调用顺序有关
3. 方法之间不能互相嵌套
4. `void`不能和`return 结果;`共存，但是可以与`return;`共存
+ `void`：代表没有返回值
+ `return 结果`：代表有返回值，先将结果返回，然后结束方法
+ `return`：仅仅代表结束方法，不代表有返回值
5. 一个方法中不能连续写多个`return`，也就代表一个方法不能有多个返回值

# 方法的重载（Overload）
1. 方法重载的定义：方法名相同，参数列表不同的方法
2. 什么叫参数列表不同：
+ 参数个数不同
+ 参数类型不同
+ 参数类型顺序不同
3. 判断两个方法是否为重载方法，和什么无关：
+ 和参数名无关
+ 和返回值无关
4. 什么时候使用：如果要定义的方法实现的功能一样，只不过实现的细节不一样，就可以考虑定义成重载方法
5. 使用示例

```java
package com.aurora.method;

public class methodDemo06 {
    /*
        方法的重载
     */
    public static void main(String[] args) {
        sum(10, 20);
        sum(10, 20, 30);
        sum(10, 20, 30, 40);
    }

    public static void sum(int a, int b){
        System.out.println(a+b);
    }

    public static void sum(int a, int b, int c){
        System.out.println(a+b+c);
    }
    public static void sum(int a, int b, int c, int d){
        System.out.println(a+b+c+d);
    }
}

```

# 方法参数
## 基本数据类型做方法参数传递
1. 基本类型做方法参数传递，**传递的是值，不是变量本身**，在方法中**修改的是变量的副本，变量本身不改变**
2. 示例

```java
package com.aurora.method;

public class methodDemo07 {
    /*
        基本数据类型做方法参数传递示例
     */
    public static void main(String[] args) {
        int a = 10, b = 20;
        method(a, b);
        System.out.println(a);  //10,不变
        System.out.println(b);  //20,不变
    }

    public static void method(int a, int b){
        a += 10;
        b += 20;
        System.out.println(a);  //20
        System.out.println(b);  //40
    }
}

```

![](https://cdn.nlark.com/yuque/0/2025/png/38867830/1758252783158-5a1db521-4861-4c03-b306-5af435a2e036.png)

## 引用数据类型做方法参数传递
1. 引用数据类型做方法参数传递时，**传递的时地址值，在变量中会修改变量本身**，因为传递的是地址，所以方法中修改的参数值该地址指向的数据，与传入的参数地址相同，是同一份数据，而不是副本
2. 示例

```java
package com.aurora.method;

public class methodDemo08 {
    /*
        引用数据类型做方法参数传递
     */
    public static void main(String[] args) {
        int[] arr = {10, 20};
        method(arr);
        System.out.println(arr[0]);     //20,本身被修改
        System.out.println(arr[1]);     //40,本身被修改
    }

    public static void method(int[] arr){
        arr[0] += 10;
        arr[1] += 20;

        System.out.println(arr[0]); //20
        System.out.println(arr[1]); //40
    }
}

```

# 命令行参数
概念：通过命令行给`main`方法的形参传递的实参成为命令行参数

```java
package com.aurora.method;

public class CommandParamDemo {
    public static void main(String[] args) {
        for (int i = 0; i < args.length; i++) {
            System.out.println("第" + (i + 1) + "个参数的值是" + args[i]);
        }
    }
}

```

1. 编译`java`程序

```powershell
javac -encoding UTF-8 com\aurora\method\CommandParamDemo.java
```

编译成功后，会得到对应的`.class`文件

```powershell
com\aurora\method\CommandParamDemo.class
```

2. 运行`java`程序

```powershell
java -cp . com.aurora.method.CommandParamDemo 1 2 3
```
