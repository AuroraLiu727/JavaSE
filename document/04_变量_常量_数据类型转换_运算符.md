# 常量
1. 定义：在代码的运行过程中，值不会发生改变的数据
2. 分类：
+ 整数常量：所有整数
+ 小数常量：所有带小数点的，例如`2.5 1.5 2.0`
+ 字符常量：写在`''`中，且只能有一个内容，例如`'1'`,`'a'`,`' '`(空格),`'    '`(tab键)，而`'11'`,`'a1'`这种不算
+ 字符串常量：带双引号`""`的，双引号中内容随意，例如`"Hello World!"`
+ 布尔常量：`true`(真)，`false`(假)
+ 空常量：`null`，表示数据不存在

```java
package com.aurora.DataType_Operation;

public class ConstantDemo01 {
    /*
        常量的学习
     */
    public static void main(String[] args) {

        // 整数常量
        System.out.println(1);
        System.out.println(-1);

        // 小数常量
        System.out.println(1.5);
        System.out.println(1.0);

        // 字符常量
        System.out.println('1');
        System.out.println('a');
        System.out.println(' ');

        // 字符串常量
        System.out.println("Hello World!");

        // 布尔常量
        System.out.println(true);
        System.out.println(false);

        // 空常量，不能直接使用
        //System.out.println(null);
    }
}

```

## 变量的介绍及使用
1. 定义：在运行过程中，值会发生变化的数据
2. 变量的数据类型：
+ 基本数据类型：4类8种
    - 整形：`byte short int long`
    - 浮点型：`float double`
    - 字符型：`char`
    - 布尔型：`boolean`
+ 引用数据类型：类  数组  接口  枚举  注解
3. 定义
+ `数据类型 变量名 = 值`
+ `数据类型 变量名;`->`变量名 = 值;`
+ 连续定义多个相同类型的变量：
    - `数据类型 变量名1, 变量名2, 变量名3;`->`变量名1 = 值, 变量名2 = 值, 变量名3 = 值;`
    - `数据类型 变量名1 = 值, 变量名2 = 值, 变量名3 = 值;`
4. 注意事项
+ 字符串不属于基本数据类型,属于引用数据类型,用String表示，String是一个类,只不过字符串在定义的时候可以和基本数据类型格式一样
+ 在开发时尽量不使用float和double型数据直接参与运算，因为可能会导致精度丢失问题
+ 定义`long`型变量时，需要在值后面加一个`L/l`，例如`long num = 10L;`
+ 定义`float`型变量时，需要在值后面加一个`F/f`，例如`long num = 2.5F;`
+ 整形的默认类型时`int`，小数的默认类型是`double`
5. 使用方法示例

## 变量使用时的注意事项

# 标识符
1. 定义：给类、方法、变量取的名字
2. 规则
+ 硬性规定(必须遵守)
    - 标识符可以包含"英文字母","数字","$和_"
    - 标识符不能以数字开头  int i1 = 100(正确)  int 1i = 100(错误)
    - 标识符不能是关键字  int static = 100(错误)   int public = 100(错误)
+ 软性建议(可遵守可不遵守,但是建议遵守)
    - 给类取名字:遵循大驼峰式 -> 每个单词首字母大写
    - 给方法和变量取名字:遵循小驼峰式 -> 从第二个单词开始往后首字母大写

# 数据类型转换
1. 什么时候发生类型转换:
+ 等号左右两边类型不一致
+ 不同类型的数据做运算
2. 转换顺序：基本类型中按照取值范围从小到大排序:`byte,short,char -> int -> long -> float -> double`  

## 自动类型转换
1. 分类：
+ 小自动转大：将取值范围小的数据类型赋值给取值范围大的数据类型
+ 小自动转大：取值范围小的数据类型和取值范围大的数据类型数据做运算
2. 示例

```java
package com.aurora.DataType_Operation;

public class TypeReverseDemo01 {
    /*
        自动类型转换
     */
    public static void main(String[] args) {
        //赋值
        float a = 1.55f;
        double b = a;   //发生自动类型转换
        System.out.println(a);
        System.out.println(b);

        //运算
        double c = 2.3;
        double sum = a + c;     //发生自动类型转换, double + float = double
        System.out.println(sum);
    }
}

```

2. 强转的方法：`取值范围小的数据类型 变量名 = (取值范围小的数据类型)取值范围大的数据类型数据`

### 自增自减运算符
1. 形式

| 格式 | 含义 |
| --- | --- |
| 变量++ | 后自增 |
| ++变量 | 先自增 |
| 变量-- | 后自减 |
| --变量 | 先自减 |


2. 使用方法
+ 单独使用：符号在前在后都是先运算
+ 混合使用：`++`、`--`和其他语句混合使用
    - 符号在前：先运算，再使用运算后的值，例如：`++b + a`
    - 符号在后：先使用原值运算，再自身运算，`b++ + a`
3. 示例

```java
package com.aurora.DataType_Operation;

public class OperatorDemo03 {
    /*
        自增自减运算符的使用
     */
    public static void main(String[] args) {
        int i = 10;
        i++;
        System.out.println("i = " + i);

        int j = 10;
        ++j;
        System.out.println("j = " + j);

        int x = 10;
        x--;
        System.out.println("x = " + x);

        int y = 10;
        y--;
        System.out.println("y = " + y);

        int z = 10;
        z = z++;
        System.out.println("z = " + z);

        int t = 10;
        t = ++t;
        System.out.println("t = " + t);
    }
}

```

3. **例子：**小明考完试了，判断小明的分数是否及格
