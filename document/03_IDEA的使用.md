# 基本语句生成快捷键
1. 生成`main`方法，输入`main`->回车
2. 生成输出语句：`sout`->回车
3. 将变量名放到输出语句中：
+ `变量名.sout`
+ `变量名.soutv`：带字符串拼接方式的输出方式，输出格式好看
